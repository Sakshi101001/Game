# 🪨 Stone Paper Scissors Game

A simple and interactive **Stone-Paper-Scissors** game built using **HTML, CSS, and JavaScript**.  
Play against the computer and see who wins!

---

## 🚀 Features
- Interactive gameplay with instant results
- Random computer choices
- Score tracking system
- Responsive and beginner-friendly UI
- Simple logic using JavaScript

---

## ⚙️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 🎮 How to Play
1. Choose **Stone**, **Paper**, or **Scissors**.
2. The computer will make a random choice.
3. The winner is decided based on the rules:
   - Stone beats Scissors  
   - Scissors beats Paper  
   - Paper beats Stone
